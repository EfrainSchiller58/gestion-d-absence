# Gestion d'Absence - School Absence Management System

A complete web-based school absence management system built with Laravel and React.

## Features
- Role-based access (Admin, Teacher, Student)
- Absence tracking and management
- Justification system with file uploads
- Real-time statistics and dashboards

## Tech Stack
- Backend: Laravel 10+
- Frontend: React 18 + Vite
- Database: MySQL
- Authentication: Laravel Sanctum
- Styling: Tailwind CSS
